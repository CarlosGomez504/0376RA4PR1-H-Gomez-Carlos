# 0376RA4PR1 - Práctica de Manipulación del DOM
**Alumno:** Carlos Gómez

Este proyecto consiste en la resolución de ejercicios prácticos sobre el DOM, aplicando estilos, eventos y modificación de la estructura HTML.

## Ejercicios de la práctica:
* **Selección:** Uso de `querySelector` para cambiar el color del H1.
* **Contenido:** Modificación del texto de un párrafo con `textContent`.
* **Atributos:** Cambio dinámico de la imagen mediante `setAttribute`.
* **Clases:** Implementación de `classList.toggle` para activar/desactivar estilos.
* **Eventos:** Uso de `addEventListener` para lanzar alertas.
* **Estructura:** Creación de nuevos elementos `li` con `createElement` y `appendChild`.
* **Eliminación:** Borrado de elementos del DOM con el método `remove()`.

## Soluciones destacadas (Código en línea):
1. Para cambiar el texto: `p1.textContent = "Hola Món";`
2. Para añadir un ítem a la lista: `llistaCompra.appendChild(nouitem);`
3. Para eliminar el elemento: `cuadroColorSalmon.remove();`

## Notas para mi sobre la sintaxis Markdown utilizada:
* **`#`**: Define un título de nivel 1 (equivalente a `<h1>`).
* **`##`**: Define un título de nivel 2 (equivalente a `<h2>`).
* **`**texto**`**: Se utiliza para poner el texto en **negrita**.
* **`*`**: Crea una lista desordenada (viñetas).
* **`` `código` ``**: Se utiliza para resaltar fragmentos de código en línea.
* **`---`**: Crea una línea horizontal de separación.
